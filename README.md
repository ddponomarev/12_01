### Задание 2

Проведите сканирование Metasploitable в режимах SYN, FIN, Xmas, UDP.

Запишите сеансы сканирования в Wireshark.

Ответьте на следующие вопросы:

- Чем отличаются эти режимы сканирования с точки зрения сетевого трафика?
- Как отвечает сервер?

*Приведите ответ в свободной форме.*

```
SYN — отправляет TCP-пакет с флагом SYN. На открытый порт сервер отвечает SYN+ACK, после чего отправляет RST, обрывая соединение. На закрытый порт сервер сразу отвечает RST.

FIN — отправляет TCP-пакет с флагом FIN. На открытый порт сервер не отвечает. На закрытый порт сервер отвечает RST.

XMAS — отправляет TCP-пакет с тремя флагами одновременно (FIN, PSH, URG). Принцип тот же, что у FIN: открытые порты молчат, закрытые отвечают RST.

UDP-сканирование — отправляет UDP-пакет. На открытый порт может прийти UDP-ответ или ничего. На закрытый порт сервер отвечает  "Port Unreachable".

```
